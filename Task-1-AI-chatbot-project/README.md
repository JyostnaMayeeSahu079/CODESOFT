# AI Rule-Based Chatbot 🤖

A full-stack AI chatbot application developed using **React.js**, **Node.js**, and **Express.js** that provides intelligent rule-based responses based on user input. The chatbot can answer predefined questions such as greetings, basic conversations, and simple queries in real time.

## Features

* Full-stack chatbot application
* Rule-based intelligent responses
* React frontend with modern UI
* Node.js & Express backend
* Real-time message handling
* REST API communication
* User-friendly chat interface
* Responsive design

## Technologies Used

### Frontend

* React.js
* CSS
* Axios

### Backend

* Node.js
* Express.js
* CORS
* Body-parser

## How It Works

1. User enters a message in the chat interface.
2. Frontend sends the message to the backend API.
3. Backend processes the message using predefined rules.
4. Bot response is returned to the frontend.
5. Chat response is displayed instantly.

## Example Questions

* Hello
* How are you?
* What is your name?
* Bye

## Future Enhancements

* AI integration using OpenAI/Gemini API
* Voice assistant support
* Database integration

## Project Objective

The main objective of this project is to understand full-stack development and chatbot implementation using rule-based logic with frontend-backend communication.
